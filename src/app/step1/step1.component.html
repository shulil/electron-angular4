Welcome to vmGuardian 4

In order to begin using the vmGuardian, you will have to configure following items.

1

backup Source

vmGuardian can backup VMware ESXi servers and Microsoft Hyper-V Servers. You must configure at least one host as backup source before you can proceed with backups

2

Destination

vmGuardian can store backup to Direct Attached Storage(connected locally) or Network Attached Storage(either via CIFS/Samba or NFS). You must configure at least one destination storage device before you can proceed with backups.

This destination storage device will be your default storage device. All Machines detected in Step 1 will be temporarily bound to this storage device. You can change association between Virtual Machine and Storage device later in the "Storage" tab.

You can register additional Backup Sources and Destination Storage, perform manual backup or scheduled backup after completing this wizard.
