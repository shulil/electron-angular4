vmGuardian can backup Virtual Machines either in Sequential Backup or Parallel Backup modes.

Sequential Backup groups as many Virtual Machines as possible and processes in Sequential Order, each subsequent backup sharing same deduplication database. Sequential backup produces one image file(containing all grouped VMs) per backup.

Parallel Backup will instead backup as many Virtual Machines as possible in simultaneously, maximizing the bandwidth offered by the source, destination, and the network. Parallel backup will create separate image files per VM , and is more replication friendly.
